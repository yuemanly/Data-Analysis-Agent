# 版本开发日志

## v1.0
- 项目正式发布第一个稳定版本
- 支持基础的数据上传、查看、分析和可视化
- 支持 Excel、CSV、数据库等常见数据源
- 可通过对话方式完成常见数据分析任务
- 提供基础的图表生成与结果导出能力

---

## v2.0
**2026年5月11日**

本次升级主要围绕 **更强的分析能力、更一致的展示体验、更好的导出能力** 展开。

### 1. 日志系统升级
- 新增统一日志系统
- 自动记录运行过程、工具调用、模型请求与异常信息
- 日志按天轮转，保留 30 天，便于排查问题

### 2. 配色方案统一
- 修复图表和 PPT 配色不一致的问题
- 用户切换品牌配色后，图表和报告会同时生效
- 默认配色为 McKinsey，也支持 BCG 等方案

### 3. Agent 架构重构
- 将原本单文件的 Agent 拆分为多个模块
- 提升代码可维护性和扩展性
- 后续新增功能会更容易接入

### 4. 国际化支持
- 新增中英双语切换
- 前端界面可一键切换中文 / English
- 页面文案、提示、命令说明等已统一适配

### 5. 斜杠命令优化
- 将原来的通用分析命令拆分为更明确的命令
- 如：`/decile`、`/tree`、`/kmeans`
- 用户更容易理解每个命令的用途

### 6. 导出功能增强
- 新增 Excel 和 Word 报告导出
- 支持将分析结果直接生成文件并下载
- `/export` 和 `/report` 命令可快速完成导出任务

### 7. 数据清洗功能增强
- 新增数据概览、缺失值处理、缩尾、截尾等清洗能力
- 支持更灵活的数据预处理流程
- 清洗后数据可直接用于后续分析

### 8. 数据预览升级
- 原来的数据结构查看改为表格预览
- 更接近 Excel 的查看体验
- 可快速浏览各个表、字段和样本数据

### 9. Token 用量追踪
- 新增上下文窗口与输出 token 统计
- 可在界面中查看当前 token 使用情况
- 便于控制模型成本和上下文长度

### 10. 思考模式展示
- 支持展示模型的推理过程
- 在需要时可查看模型思考链
- 适合复杂分析任务的调试与理解

### 11. 前端体验优化
- 页面结构进一步拆分，维护更清晰
- 样式与脚本独立管理
- 界面交互更稳定，响应更流畅
