# 点密度地图 Dot Density Map

图表分类: 地理 Geospatial  
感知排名: ⭐⭐⭐⭐☆

## 简介

点密度地图用散点在地理坐标系上表示数据分布：
- 散点着色：第一列数值（如人口密度）→ 散点颜色深浅
- 散点位置：预定义的地区中心坐标
- 交互：支持缩放、拖拽、悬停显示地区名

麦肯锡配色梯度，颜色深浅表示数值大小。

## 数据格式

需要包含以下字段：

- city：市名称（如"宜昌"）
- label：地区名称（如"夷陵区"、"西陵区"）
- value：数值列（用于散点着色，如人口密度）

**示例数据：**

| 市 | 地区 | 人口密度 |
|----|------|---------|
| 宜昌 | 夷陵区 | 163.73 |
| 宜昌 | 西陵区 | 5934.44 |
| 宜昌 | 伍家岗区 | 4897.10 |
| 宜昌 | 枝江市 | 327.33 |
| 宜昌 | 当阳市 | 193.93 |

## 适用场景

点密度地图适用于以下场景：

- 人口密度地理分布
- 经济指标地理分布
- 任何需要展示"地理位置 + 数值大小"的分析

## 典型场景

- 城市各区人口密度分布
- 各省 GDP 密度分布
- 小区居住密度分布
- 商业热度地理分布

## 使用建议

- 数值列建议使用密度/比率类指标（适合着色）
- 地区名需与预定义坐标表匹配
- 如果地区名无法匹配，该地区的散点会被跳过

## 与相似图表的区别

### 点密度图 vs 面量图

| 对比维度 | 点密度图 | 面量图 |
|---------|---------|--------|
| 视觉编码 | 散点颜色深浅 | 区域填充颜色深浅 |
| 数据表达 | 单一数值 | 单一数值 |
| 常见用途 | 人口密度、经济指标 | 人口密度、经济指标 |
| 交互 | ✅ 支持缩放拖拽 | ✅ 支持缩放拖拽 |
| 精度 | 高（点级别） | 中（区域级别） |

## 注意事项

- 地区名需与预定义坐标表匹配（如"宜昌市"→"宜昌"）
- 如果地区名无法匹配，预定义坐标表中没有该地区时散点会被跳过
- 悬停显示地区名称
