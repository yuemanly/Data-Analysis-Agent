# 金字塔图

图表分类: 分布类 DISTRIBUTION
感知排名: ⭐⭐⭐⭐☆

## 简介

金字塔图通过在中心竖轴的两侧展示两个群体的数据，使用户可以快速观察：

- 人口年龄结构分布
- 两个群体（如男性和女性）的相对规模
- 整体分布形状（如人口老龄化程度）
- 不同年龄段的人口比例差异

相比并排的柱状图，金字塔图能更直观地展示整体分布形状，便于识别人口结构特征。


## 数据格式

需要包含以下字段：

- Label（必需）：例如 年龄段标签（如"0-4岁"、"5-9岁"等）
- Group1（必需）：例如 男性数值（人数、百分比等）
- Group2（必需）：例如 女性数值（人数、百分比等）

示例：

| 年龄段 | 男性 | 女性 |
|--------|------|------|
| 0-4岁 | 12500 | 12000 |
| 5-9岁 | 13200 | 12800 |
| 10-14岁 | 14100 | 13500 |
| 15-19岁 | 15000 | 14200 |
| 20-24岁 | 16500 | 15800 |


## 适用场景

- 人口年龄结构分析
- 性别比例分布展示
- 人口老龄化程度评估
- 不同国家/地区人口结构对比
- 历史人口变化趋势展示


## 典型场景

- 国家人口金字塔（按年龄和性别分布）
- 城市人口结构分析
- 企业员工年龄分布
- 学生群体性别和年级分布
- 医疗人口统计数据展示


## 使用建议

- 数据应按年龄段从小到大排序，便于形成金字塔形状
- 男性数据会自动显示在左侧（负值），女性数据显示在右侧（正值）
- 可使用绝对数值（人数）或相对数值（百分比）
- 年龄段标签应清晰简洁（如"0-4"、"5-9"等）
- 当年龄段过多（>30）时，考虑合并相邻年龄段


## 注意事项

- 数据必须包含 age_group、male、female 三列
- 系统会自动将男性数据转换为负值显示在左侧
- 女性数据保持正值显示在右侧
- 悬停提示会显示实际数值（不含负号）
- 图例位置在图表下方，便于查看


## 与相似图表的区别

### 金字塔图 vs 并排柱状图

- 金字塔图：左右对称展示，便于观察整体分布形状
- 并排柱状图：并排显示，便于精确比较两个群体的数值

### 金字塔图 vs 点图

- 金字塔图：用柱子展示数值大小，适合展示连续的年龄段
- 点图：用点和线展示，适合展示离散的类别对比

### 金字塔图 vs 堆积柱状图

- 金字塔图：左右对称，展示两个群体的分布
- 堆积柱状图：上下堆积，展示多个类别的组成部分
