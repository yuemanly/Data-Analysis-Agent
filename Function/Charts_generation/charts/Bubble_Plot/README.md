# 气泡图 Bubble Chart

图表分类: 关系类 RELATIONSHIP
感知排名: ⭐⭐⭐⭐☆

## 简介

气泡图（ Bubble Chart ）是散点图的增强版本，通过**气泡的横纵坐标（x/y）、大小（size）、颜色（color）** 四个维度联动展示数据，适合用于：

- 多维度关系分析（相关性、聚类、异常值识别）
- 分组对比（不同类别用不同颜色区分）
- 四象限策略分析（通过 x_mid/y_mid 参数叠加象限分界线）

## 数据格式

### 必需字段

| 角色 | 类型 | 说明 |
|---|------|------|
| x | 数值列 | 自变量（横轴） |
| y | 数值列 | 因变量（纵轴） |

### 可选字段

| 角色 | 类型       | 说明                     |
|-----|----------|------------------------|
| size | 数值列      | 控制气泡大小（映射到像素直径，18–80px） |
| x_mid | 数值列      | X轴象限分界线                |
| y_mid| 数值列 | Y轴象限分界线                |
|color|类别列（非数值）|	控制气泡颜色|


### 宽格式支持

不支持宽格式。所有数据必须为长格式，每行代表一个数据点。
推荐数据结构如下：
| 业务单元 | 市场渗透率（%） | 利润率（%） | 营收规模（亿元） | x 轴中线 | y 轴中线 |Color |
|----------|:--------------:|:----------:|:---------------:|:--------:|:--------:|:--------:|
| 业务A | 85.2 | 42.5 | 18.7 | 50 | 30 |-30 |
| 业务B | 72.1 | 35.8 | 12.3 |  |  |100 |
| 业务C | 91.3 | 28.2 | 25.1 |  |  |-50 |
| 业务D | 95.8 | 22.7 | 8.9 |  |  |30 |
| 业务E | 88.6 | 15.3 | 5.6 |  |  |60 |
| 业务F | 35.4 | 45.1 | 3.2 |  |  |80 |

## 适用场景

- 相关性分析：如"消费倾向 vs 渗透率"、"净利前 vs 盈利"
- 四象限策略分析：叠加 x_mid/y_mid 参数生成象限分界线，用于矩阵式战略定位（如 Bain 聚类结果可视化）
- 分组聚类：用 color 参数按类别着色，识别不同组别在 x/y 空间的分布特征
- 异常值识别：气泡大小或颜色异常的数据点一目了然

## 典型场景

1. **相关性探索**：探索两个数值变量之间的相关性（价格 vs 销量、GDP vs 碳排放）
2. **四象限战略定位**：设定 x/y 分界线，将城市/产品划分为四个象限（如高规模/高盈利结构 vs 低规模/低盈利结构）
3. **分组对比**：按地区/类别着色，对比不同组别在二维空间的分布差异
4. **异常值检测**：气泡大小或颜色异常的点即为需要重点关注的数据

## 使用建议

- 气泡数量控制在 5–30 个，过多会导致重叠和信息过载
- x/y 轴范围自动扩展 6%，避免气泡与轴线重叠
- 颜色映射优先识别类别列（字符串列），若传入数值列则按数值大小着色
- 若气泡存在重叠，可对离散数据添加微小的随机扰动（jitter）

---

## 注意事项

- x 和 y 必须为连续数值列，不可为类别列
- 若未指定 size，所有气泡默认为 40px（中等尺寸）
- 若未指定 color，所有气泡统一使用默认主色
- 象限线 x_mid/y_mid仅在明确传入时才会绘制，为可选功能
- 当x范围在 0–1 之间时，轴标签自动乘以 100 显示为百分比形式

## 与相似图表的区别

### 散点图（Scatter Plot）vs 气泡图

- 散点图：仅展示 x/y 两个维度
- 气泡图：额外通过大小和颜色增加两个维度，适合多维数据探索

### 气泡图 vs 气泡矩阵

- 气泡图：通过 `color` 参数自动识别类别着色，适合一般性多维度分析
- 气泡矩阵：通常指定 x/y/size 三个维度，颜色作为第三维度的编码方式（如盈利状况），可通过象限线做战略定位

### 气泡图 vs 分组柱状图

- 分组柱状图：适合类别对比，x轴为类别变量
- 气泡图：适合连续变量之间的相关性分析，x轴和y轴均为连续变量
