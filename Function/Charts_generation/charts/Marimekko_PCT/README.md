# 马赛克图（百分比版）

图表分类: 对比 COMPARING  
感知排名: ⭐⭐⭐⭐☆

## 简介

马赛克图百分比版（Marimekko PCT）通过改变柱宽和柱内分段高度同时编码两个维度的数据：

- 柱宽表示第一维度的相对占比（如品牌总销售额占比）
- 柱内堆叠高度表示第二维度的相对占比（如各产品类别的销售占比）
- 矩形面积同时反映两个维度的综合信息
- 适合展示"大类别中的小分组"的相对占比关系

相比堆叠柱状图，马赛克图能更直观地展示各类别的相对规模差异。

## 数据格式

需要包含以下字段：

- x：品牌/类别列，文本类型（必需）
- y：销售额/数值列，数值类型（必需）
- group：产品类别/分组列，文本类型（可选）

### 长格式数据（有分组）

| 品牌  | 2020 | 2021 | 2022 |
|-----|------|------|------|
| 品牌A | 300  | 200  | 250 |
| 品牌B | 150  | 200  | 120 |
| 品牌C | 200  | 150  | 320 |
| 品牌D | 400  | 50   | 360 |

**处理规则：**
- 柱宽 = 品牌总销售额 / 全部总销售额（相对占比）
- 柱内高度 = 各产品类别销售额 / 该品牌总销售额（相对占比）

## 适用场景

马赛克图百分比版适用于以下场景：

- 展示多个品牌的相对规模和各品牌内产品的相对占比
- 比较不同类别中的子分组相对占比
- 分析部分与整体的相对关系（两个维度都用占比表示）
- 展示市场份额和细分市场的相对占比

## 典型场景

- 品牌销售额分布及各品牌的产品类别比例
- 公司部门规模及各部门的成本构成
- 产品线销售额及各产品在不同地区的分布
- 行业市场份额及各企业的产品线分布

## 使用建议

- 品牌/类别数建议 3-8 个，过多会导致图表拥挤
- 柱宽自动计算为各品牌的相对占比（总和=100%）
- 柱内高度表示该品牌内各分组的相对占比（总和=100%）
- 如有分组，会自动堆叠显示
- 颜色自动按麦肯锡配色方案分配
- 适合对比不同品牌的内部构成差异

## 注意事项

- 感知难度较高，需要用户理解宽度和高度都表示占比的含义
- 分组过多（>5个）会导致颜色难以区分
- 建议配合图例和数据标签使用
- 柱宽和柱内高度都是相对占比，便于对比相对构成
- 两个维度都用百分比表示，适合展示相对关系

## 与相似图表的区别

### 马赛克图（百分比版）vs 马赛克图（绝对值版）

- 百分比版：柱内高度表示各分组的相对占比，便于对比相对构成
- 绝对值版：柱内高度表示各分组的绝对销售额，便于对比实际数值

### 马赛克图 vs 堆叠柱状图

- 马赛克图：柱宽表示品牌占比，柱内高度表示分组占比，双重编码
- 堆叠柱状图：柱宽固定，只用高度表示数值

### 马赛克图 vs 分组柱状图

- 马赛克图：柱宽可变，展示品牌相对规模
- 分组柱状图：柱宽固定，便于精确比较各分组的数值
