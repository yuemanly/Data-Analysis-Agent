# 华夫格 Waffle Chart

图表分类: 对比类 COMPARING  
感知排名: ⭐⭐⭐☆☆

## 简介

华夫格是一种单位图表，通过 10×10 网格（共 100 个单元格）直观展示各分类占整体的比例。每个单元格代表 1 个百分点，特别适合演示和演讲场景。

它通过**色块填充比例**，使用户可以：

- 快速感知各分类的相对占比
- 直观理解"部分占整体"的关系
- 在演讲中引起观众共鸣（相比数字更直观）

相比**柱状图**，华夫格更强调"占比"而非"绝对值"；相比**饼图**，华夫格的精度更高、更易比较。


## 数据格式

需要包含以下字段：

- label（类别）：文本类型，代表各分类的名称（如"产品A"、"产品B"）
- value（数值）：数值类型，代表各分类的数量或占比（必须 ≥ 0）

### 数据示例

| 产品 | 销售额 |
|------|--------|
| 产品A | 250 |
| 产品B | 180 |
| 产品C | 120 |
| 产品D | 100 |
| 产品E | 50 |

### 宽格式数据

华夫格**不支持**宽格式数据转换。请确保数据为长格式（每行一个类别）。


## 适用场景

华夫格适用于以下场景：

- 市场份额分布 — 展示各品牌/产品的市场占有率
- 投票/问卷结果 — 展示各选项的投票比例
- 预算分配 — 展示各部门的预算占比
- 客户分布 — 展示各地区/渠道的客户占比
- 演讲/演示 — 需要直观、易理解的占比展示
- 小样本占比 — 类别数 ≤ 6 时效果最佳


## 典型场景

- 电商平台：展示各品类销售额占比（如服装 35%、电子 28%、食品 20%）
- 市场调研：展示消费者对不同品牌的偏好占比
- 财务报表：展示各成本项目占总成本的比例
- 用户分布：展示不同用户等级的占比（VIP 15%、普通 85%）
- 项目进度：展示已完成/进行中/未开始的任务占比


## 使用建议

- 类别数控制：建议 2-6 个类别，最多不超过 10 个
- 数值要求：所有数值必须 ≥ 0，总和 > 0
- 标签清晰：类别名称应简洁明了，避免过长
- 百分比显示：图例自动显示各分类的百分比，无需手动计算


## 注意事项

- 精度限制：华夫格的精度低于柱状图，不适合精确数值比较
- 类别过多：超过 6 个类别时，建议改用柱状图或其他图表
- 小数值处理：极小的数值（< 1%）可能无法显示为单个单元格
- 响应式设计：在小屏幕上可能显示不佳，建议在桌面端查看


## 与相似图表的区别

### 华夫格 vs 柱状图

- 华夫格：强调占比关系，直观易懂，适合演讲
- 柱状图：强调绝对值，精度高，适合数据分析

### 华夫格 vs 饼图

- 华夫格：精度更高，易于比较相近的占比
- 饼图：传统直观，但精度低，难以比较相近的切片

### 华夫格 vs 堆叠柱状图

- 华夫格：单一维度占比，适合演示
- 堆叠柱状图：多维度对比，适合分析趋势

### 华夫格 vs 百分比条形图

- 华夫格：网格形式，视觉冲击力强
- 百分比条形图：线性形式，易于精确读数
