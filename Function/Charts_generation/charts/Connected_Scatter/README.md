# 连线散点图 Connected_Scatter

图表分类: 时间趋势类 TIME 
感知排名: ⭐⭐⭐⭐☆

## 简介

连线散点图在散点图的基础上，用线段连接各点展示数据的演变过程或轨迹。通过连接线和标记点的组合，直观观察数据点之间的演变过程、因果关系或时间序列的发展趋势。

相比散点图，连线散点图强调有序的演变过程；相比折线图，连线散点图更适合展示非时间序列的关系演变，且点的大小可表示第三维度。

## 数据格式

需要包含以下字段：

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| x | 数值 | ✅ | X轴数据，通常为某个指标或维度 |
| y | 数值 | ✅ | Y轴数据，通常为另一个指标或维度 |
| order | 数值/文本/日期 | ✅ | 排序列，决定点的连接顺序（年份、时间、序列号等） |
| size | 数值 | ❌ | 气泡大小列，表示第三个维度（可选） |

## 适用场景

- 产品演变过程：展示产品从推出到成熟的销售额和利润变化
- 公司发展轨迹：展示公司在不同时期的收入和成本关系
- 学生成长过程：展示学生在不同阶段的成绩和排名变化
- 市场演变分析：展示市场份额和价格的演变关系
- 国家发展指标：展示GDP与寿命、教育等指标的演变轨迹
- 股价演变：展示股票价格与成交量的演变过程

## 典型场景

- 销售与利润演变：展示5年间销售额与利润的相关性变化
- 股价与成交量：展示股票价格与成交量的演变轨迹
- 用户增长与留存：展示新增用户与留存率的演变过程
- 成本与效率：展示生产成本与生产效率的演变关系

## 使用建议

- 数据点建议 ≥ 3 个，以展示明显的演变过程
- order列必须提供，确保连接顺序正确
  - 支持数值排序（年份、序列号）
  - 支持日期排序（自动识别）
  - 支持字符串排序（按字典序）
- size列可选，用于表示第三维度
  - 自动线性映射到8-40px范围
  - 如不提供，所有点大小统一为15px
- 配色方案支持McKinsey、BCG、Bain等专业配色

## 注意事项

- 缺失值会被自动删除，可能导致连接线中断
- x和y列必须可转换为数值类型
- order列用于确定连接顺序，缺少会导致图表混乱
- 少于2个有效点会导致生成失败
- size列会自动线性缩放到8-40px范围内

## 与相似图表的区别

### 连线散点图 vs 散点图

- 连线散点图：点之间有连接线，强调演变过程和路径
- 散点图：仅显示点，强调点的分布和相关性

### 连线散点图 vs 折线图

- 连线散点图：适合非时间序列的关系演变，点的大小可表示第三维度
- 折线图：适合时间序列数据，强调时间趋势

### 连线散点图 vs 气泡图

- 连线散点图：强调演变过程，点之间有连接线
- 气泡图：强调三维关系，点的大小表示第三维度，无连接线
