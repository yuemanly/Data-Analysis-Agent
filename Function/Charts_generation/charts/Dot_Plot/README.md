# 点图 Dot Plot

图表分类: 比较 Comparisons  
感知排名: ⭐⭐⭐⭐⭐

## 简介

点图（Dot Plot）由数据可视化先驱 William Cleveland 开发，是展示**范围、差异或变化**的轻量级图表。它用圆点代替柱形，通过连接线展示两个数据点之间的关系，相比柱状图使用更少的"墨水"，视觉更清爽。

它通过**圆点 + 连接线**，使用户可以：

- 快速比较多个类别的范围或差异 — 如起点和终点的距离
- 清晰识别变化方向 — 通过连接线的方向和长度
- 在类别众多时保持清爽 — 避免柱状图的拥挤感
- 展示两个时间点的变化 — 如 2015 年 vs 2020 年的成绩

相比**柱状图**，点图更轻量、更适合多类别；相比**折线图**，点图强调**两个离散时间点**而非连续趋势。


## 数据格式

需要包含以下字段：

- 类别列（Y轴）：文本类型，代表不同的类别（如国家、产品、部门等）
- 起点列（X轴）：数值类型，代表第一个数据点（如 2015 年成绩、起始值）
- 终点列（X轴）：数值类型，代表第二个数据点（如 2020 年成绩、结束值）

**数据示例：**

| 国家 | 2015年数学成绩 | 2020年数学成绩 |
|------|--------|--------|
| 中国 | 613 | 625 |
| 日本 | 532 | 547 |
| 美国 | 470 | 478 |
| 英国 | 494 | 502 |


## 适用场景

点图适用于以下场景：

- 两个时间点的变化 — 展示 2015 vs 2020、去年 vs 今年的对比
- 起点和终点的范围 — 如目标值 vs 实际值、预期 vs 实际
- 多类别的差异展示 — 比较多个国家、产品、部门的变化幅度
- 避免柱状图拥挤 — 当类别数较多（20-50个）时，点图比柱状图更清爽
- 强调差异而非绝对值 — 重点关注变化量而非具体数值


## 典型场景

- 国际学生成绩对比 — 多个国家的 2015 年 vs 2020 年数学/阅读成绩
- 销售目标达成 — 目标销售额 vs 实际销售额（多个销售员/地区）
- 成本控制 — 预算 vs 实际支出（多个部门）
- 员工绩效变化 — 上年度评分 vs 本年度评分（多个员工）
- 市场份额变化 — 去年市场份额 vs 今年市场份额（多个竞争对手）


## 使用建议

- 排序优化 — 按范围大小（中点）排序，便于快速识别变化最大/最小的类别
- 类别数量 — 建议 10-50 个类别；超过 50 个时考虑分组或其他图表
- 类别标签 — 自动靠右显示，确保清晰可读


## 注意事项

- 缺失值处理 — 自动删除起点或终点为空的行
- 数据类型 — 数值列会自动转换为浮点数；类别列转换为字符串
- 类别数上限 — 超过 50 个类别时会产生警告，建议分组展示
- 中间年份数据 — 点图仅展示两个时间点，不显示中间年份的波动；如数据在中间年份有剧烈变化，应考虑使用折线图


## 与相似图表的区别

### 点图 vs 柱状图

| 特性 | 点图 | 柱状图 |
|------|------|--------|
| 视觉重量 | 轻量 | 较重 |
| 类别容纳量 | 多（20-50） | 中等（10-30） |
| 展示内容 | 两个点 + 范围 | 单一值 |
| 多系列支持 | 不支持 | 支持 |
| 适合场景 | 变化对比 | 单指标比较 |

**选择建议：** 展示两个时间点的变化 → 点图；展示单一指标的多类别 → 柱状图

### 点图 vs 折线图

| 特性 | 点图 | 折线图 |
|------|------|--------|
| 时间点数 | 2个（离散） | 多个（连续） |
| 中间数据 | 不显示 | 显示所有 |
| 用途 | 两点对比 | 趋势展示 |
| 类别数 | 多（20-50） | 少（2-5） |

**选择建议：** 仅关注两个时间点 → 点图；需要展示完整趋势 → 折线图

### 点图 vs 哑铃图

| 特性 | 点图 | 哑铃图 |
|------|------|--------|
| 连接方式 | 直线 | 通常为直线 |
| 圆点大小 | 统一 | 可变 |
| 用途 | 范围对比 | 范围 + 权重 |

**选择建议：** 简单的两点对比 → 点图；需要表示权重/大小 → 哑铃图
