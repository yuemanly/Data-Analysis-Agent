# 平行坐标图 Parallel Coordinates Plot

图表分类: 多变量 Multivariate  
感知排名: ⭐⭐⭐⭐☆

## 简介

平行坐标图用多条竖直轴表示不同的变量，每条线连接各轴上的点，展示多个变量之间的关系。与散点图只能展示两个变量不同，平行坐标图可以同时展示3个或更多变量，帮助发现多变量间的相关性、聚类和模式。

平行坐标图的轴可以有不同的单位和测量范围，也可以标准化为统一的0-1范围。通过改变轴的顺序，可以发现不同的相关性。

## 数据格式

需要包含以下字段：

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| 维度1 | 数值 | ✅ | 第一个变量 |
| 维度2 | 数值 | ✅ | 第二个变量 |
| 维度3+ | 数值 | ✅ | 第三个及以上变量（至少2个） |
| color | 文本 | ❌ | 分组列，用于颜色分组（可选） |

### 典型数据结构

| 国家     | 教育程度 | 就业率 | 寿命 | 投票率 | 地区   |
|----------|----------|--------|------|--------|--------|
| 瑞典     | 95       | 85     | 82   | 80     | 欧洲   |
| 美国     | 92       | 78     | 79   | 65     | 北美   |
| 日本     | 98       | 80     | 84   | 75     | 亚洲   |
| 巴西     | 75       | 65     | 75   | 70     | 南美   |
| 尼日利亚 | 60       | 50     | 54   | 45     | 非洲   |

**字段映射示例：**
- dimensions: ["教育程度", "就业率", "寿命", "投票率"]
- color: 地区（可选）

## 适用场景

- **多变量相关性分析**：展示教育程度、就业率、寿命、投票率等多个指标的关系
- **国家/地区发展指标**：展示不同国家在多个维度上的表现（Gapminder风格）
- **产品性能对比**：展示不同产品在多个性能指标上的表现
- **学生综合评估**：展示学生在多个科目上的成绩
- **企业财务分析**：展示企业在收入、利润、增长率等多个指标上的表现

## 典型场景

- **国际发展指标**：展示32个国家在教育程度、就业率、寿命、投票率等6个维度上的关系
- **产品对比**：展示5个竞争产品在价格、性能、可靠性、用户评分等维度上的表现
- **学生成绩**：展示班级学生在语文、数学、英语、物理、化学等科目上的成绩
- **企业财务**：展示多个季度的收入、利润、增长率、市场份额等指标

## 使用建议

- 维度数量建议3-6个，过多会导致图表混乱
- 数据行数建议10-100行，过少无法看出模式，过多会导致线条重叠
- **标准化轴**：启用normalize选项将所有轴标准化到0-1范围
  - 优点：便于比较不同单位的变量
  - 缺点：会隐藏各变量的实际范围差异
- **独立范围**：禁用normalize选项保持各轴的独立范围
  - 优点：显示各变量的实际范围差异
  - 缺点：需要更多轴标签
- **透明度处理**：使用opacity参数处理线条重叠
  - 默认0.5，可根据数据密度调整
  - 值越小越透明，越容易看到重叠
- **分组颜色**：使用color列区分不同类别
  - 自动分配不同颜色
  - 支持最多10个分组
- **轴顺序**：改变维度顺序可以发现不同的相关性

## 注意事项

- 缺失值会被自动删除
- 所有维度列必须可转换为数值类型
- 少于1行有效数据会导致生成失败
- 大量线条重叠会影响视觉效果，可考虑减少数据行数或增加透明度
- 维度过多（>6）会导致图表过于复杂，建议选择最重要的维度

## 与相似图表的区别

### 平行坐标图 vs 散点图

- 平行坐标图：展示3个或更多变量的关系，用多条竖直轴
- 散点图：展示2个变量的关系，用水平和竖直轴

### 平行坐标图 vs 热力图

- 平行坐标图：展示多变量间的相关性和路径，强调线条连接
- 热力图：展示数据矩阵的强度分布，强调颜色深浅

### 平行坐标图 vs 雷达图

- 平行坐标图：用竖直轴展示多变量，适合展示多个观测值
- 雷达图：用射线展示多变量，适合展示单个或少数几个观测值

## 高级功能

### 标准化轴

启用normalize选项将所有轴标准化到0-1范围，便于比较不同单位的变量：

```python
generate(
    df=df,
    mapping={"dimensions": ["教育程度", "就业率", "寿命"]},
    options={"normalize": True}  # 默认True
)
```

### 独立范围

禁用normalize选项保持各轴的独立范围，显示实际范围差异：

```python
generate(
    df=df,
    mapping={"dimensions": ["教育程度", "就业率", "寿命"]},
    options={"normalize": False}
)
```

### 透明度处理

使用opacity参数处理线条重叠：

```python
generate(
    df=df,
    mapping={"dimensions": ["教育程度", "就业率", "寿命"]},
    options={"opacity": 0.3}  # 0-1，默认0.5
)
```

### 分组颜色

使用color列区分不同类别：

```python
generate(
    df=df,
    mapping={
        "dimensions": ["教育程度", "就业率", "寿命"],
        "color": "地区"
    },
    options={"title": "国际发展指标"}
)
```
