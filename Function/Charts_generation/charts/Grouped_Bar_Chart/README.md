# Grouped Bar

图表分类: 对比 COMPARING 
感知排名: ⭐⭐⭐⭐⭐

## 简介

分组柱状图（Grouped Bar Chart）用于在**同一类别下比较多个分组的数值大小**。  
它通过将不同分组的柱子并排展示，使用户可以直观地观察：

- 同一类别内不同组之间的差异
- 不同类别之间的变化趋势
- 各分组在多个类别中的相对表现

相比堆叠柱状图，分组柱状图更适合强调**组间对比**，而不是总量构成。


## 数据格式

需要包含以下字段：

- x列（类别下分组）：表示横轴分类
- y列（类别）：表示对应的数值大小

推荐数据结构如下：

| 行标签 | 标签二 | 标签三 | 标签四 | 标签一 |
|--------|--------|--------|--------|--------|
| 第一周 | 36.39% | 49.72% | 73.50% | 24.53% |
| 第二周 | 31.97% | 44.48% | 69.08% | 23.86% |
| 第三周 | 29.03% | 35.79% | 63.14% | 21.95% |
| 第四周 | 28.11% | 35.32% | 60.85% | 21.24% |
| 第五周 | 25.46% | 30.22% | 56.75% | 13.82% |


## 适用场景

分组柱状图适用于以下场景：

- 比较不同类别下多个对象的数值差异
- 展示多个系列在各分类中的表现
- 适合离散分类数据的横向对比
- 强调“同类目下不同组”的比较关系


## 典型场景

- 比较不同地区在不同年份的销售额
- 对比不同产品在各渠道中的销量
- 展示不同班级在各学科中的平均成绩
- 比较不同部门在各季度的预算使用情况


## 使用建议

- 当分组数量较少（通常 2~5 个）时效果最佳
- 如果类别过多或分组过多，图表会显得拥挤，建议改用：
  - 堆叠柱状图
  - 折线图
  - 分面图
- 建议保持统一的颜色编码，便于识别不同分组
- 类别名称较长时可考虑旋转标签或改为横向条形图

## 注意事项

- x应为离散类别字段，不建议直接使用连续数值
- group 字段应为有限个分类值，否则图例和柱子数量会过多
- y应为数值型字段
- 若某些类别下缺少部分分组数据，可能会出现柱子缺失，建议在绘图前补齐数据


## 与相似图表的区别

### 分组柱状图 vs 堆叠柱状图

- 分组柱状图：适合比较各分组之间的差异
- 堆叠柱状图：适合查看总量及内部构成

### 分组柱状图 vs 基础柱状图

- 分组柱状图：一个类别中包含多个系列
- 基础柱状图：每个类别通常只有一个数值系列
