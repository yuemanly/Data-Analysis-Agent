## 简介

直方图是一个灵活的分布展示工具，支持两种模式：

- 模式 1（单列数据）：频率分布直方图，展示数值变量的分布特征
- 模式 2（双列数据）：帕累托图，结合柱状图和累积曲线，展示频次分布及其累积百分比

通过这两种模式，用户可以快速观察：
- 数据的集中趋势（众数、中位数）
- 数据的离散程度（方差、标准差）
- 分布形状（正态、偏态、双峰等）
- 累积贡献度（帕累托原则：80/20 法则）


## 数据格式

### 模式 1：单列数据（频率分布直方图）

需要包含以下字段：

- value（必需）：数值列，用于绘制直方图。建议数据点数 ≥ 30，以保证分布形状稳定

示例：

| 年龄 |
|------|
| 25 |
| 28 |
| 32 |
| 35 |
| 42 |

### 模式 2：双列数据（帕累托图）

需要包含以下字段：

- x（必需）：类别列，用于分组（如产品类别、行业、地区）
- y（必需）：数值列，用于计算频次和累积总和（如销售额、收入、频次）

示例：

| 缺陷类型   | 发生频次 |
|------------|----------|
| 表面划痕   | 120      |
| 尺寸偏差   | 85       |
| 装配错误   | 52       |
| 元件损坏   | 30       |
| 焊接不良   | 18       |
| 其他       | 13       |


## 适用场景

### 模式 1：频率分布直方图

- 频率分布分析（成绩分布、收入分布、年龄分布）
- 发现数据的偏态、双峰等形状特征
- 评估数据的正态性
- 识别异常值和离群点

### 模式 2：帕累托图

- 帕累托分析：识别 80% 的结果来自 20% 的原因
- 销售分析：展示各产品/地区的销售额及其累积贡献
- 质量管理：展示缺陷类型及其累积频率
- 客户分析：展示客户价值分布及其累积贡献
- 库存管理：展示 SKU 销量及其累积占比


## 典型场景

### 模式 1

- 员工年龄分布分析
- 销售额频率分布
- 考试成绩分布
- 客户消费金额分布
- 产品质量指标分布

### 模式 2

- 各行业收入分布与累积贡献
- 产品销售额排名与累积占比
- 缺陷类型频率与累积百分比
- 客户消费金额分布与累积价值
- 支出类别与累积占比分析


## 使用建议

### 模式 1

- 数据量建议 ≥ 30，以保证分布形状稳定
- 箱数（nbins）通常设置为 20-50，可根据数据量调整
- 对于离散数据，先进行分箱处理再绘制
- 可配合密度曲线（KDE）展示平滑的分布趋势

### 模式 2

- 数据会自动按 y 值从高到低排序，便于快速识别主要贡献者
- 累积曲线展示帕累托累积百分比，用于识别"关键少数"（80/20 法则）
- 当类别过多（>20）时，考虑合并小类别为"其他"
- 双 y 轴设计：左轴为频次，右轴为累积百分比


## 注意事项

- 系统会自动检测数据列数，选择合适的模式
- 单列数据时分布形状不稳定（<20），建议补充数据
- 箱宽/箱数选择会影响直方图的形状，需根据数据特征调整
- 离散数据（如整数）应先做分箱处理，避免出现多个孤立的柱子
- 双列数据时，累积百分比基于总和计算，范围为 0-100%


## 与相似图表的区别

### 直方图 vs 柱状图

- 直方图：用于连续数值数据，展示频率分布，x 轴是连续的数值范围
- 柱状图：用于离散类别数据，展示各类别的数值，x 轴是离散的类别

### 直方图 vs 密度图

- 直方图：离散的柱子，展示具体的频次
- 密度图：连续的曲线，展示平滑的概率密度

### 直方图 vs 箱线图

- 直方图：展示完整的分布形状
- 箱线图：展示四分位数和异常值，更紧凑

### 帕累托图 vs 堆积柱状图

- 帕累托图：展示累积百分比趋势，用于识别关键因素
- 堆积柱状图：展示各类别的组成部分
