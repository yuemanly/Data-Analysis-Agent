# 南丁格尔玫瑰图 Nightingale Rose

图表分类: 占比 Part-to-Whole  
感知排名: ⭐⭐⭐☆☆

## 简介

南丁格尔玫瑰图是一种极坐标扇形面积图，通过扇形的面积（而非半径）编码数值大小。它通过极坐标系统展示周期性或分类数据的占比关系，使用户可以：

- 快速识别周期性数据的规律（如季节性变化）
- 对比不同类别的相对大小
- 在演示场景中获得强烈的视觉冲击力

相比饼图，南丁格尔玫瑰图能展示更多类别（≤12）；相比柱状图，它具有更强的视觉冲击力，但精度较低。

## 数据格式

**必需列**

| 列名 | 类型 | 说明 |
|------|------|------|
| names | 文本/类别 | 扇区名称（如月份、类别） |
| values | 数值(≥0) | 数值大小，用于计算扇形面积 |

**示例数据**

| 月份 | 销售额 |
|------|--------|
| 1月 | 15000 |
| 2月 | 18000 |
| 3月 | 22000 |
| ... | ... |
| 12月 | 30000 |

## 硬性约束

1. **面积编码**：扇形面积 = 数值
2. **类别数≤12**（月份/季度），超过会导致视觉混乱
3. 各数值必须 ≥ 0（不支持负数）

## 适用场景

✅ **典型场景**
- 周期性数据：12个月销售额、4个季度收入、7天流量
- 历史数据可视化：按月份/季度展示历史趋势
- 演示场景：视觉冲击力强，适合报告展示
- 分类数据占比：≤12个类别的占比关系

✅ **使用建议**
1. 类别数控制在 4-12 个为佳
2. 数据应按时间或逻辑顺序排列
3. 配合标题和图例清晰表达含义
4. 选择合适的配色方案增强视觉效果

## 注意事项

- 类别过多（>12）会导致视觉混乱，建议用柱状图
- 数值必须 ≥ 0（负数应使用其他图表）
- 人类对极坐标的感知精度低于笛卡尔坐标，不适合精确比较

## 与相似图表的区别

### 南丁格尔玫瑰图 vs 饼图

| 特性 | 南丁格尔玫瑰图 | 饼图 |
|------|--------|------|
| 类别数 | 多（4-12） | 少（2-5） |
| 编码方式 | 面积 | 角度 |
| 精度 | 中 | 低 |
| 视觉冲击 | 强 | 中 |
| 适用 | 周期数据 | 简单占比 |

### 南丁格尔玫瑰图 vs 柱状图

| 特性 | 南丁格尔玫瑰图 | 柱状图 |
|------|--------|--------|
| 类别数 | 4-12 | 3-30 |
| 精度 | 中 | 高 |
| 视觉冲击 | 强 | 中 |
| 适用 | 周期/演示 | 通用对比 |
| 可读性 | 中 | 高 |

### 南丁格尔玫瑰图 vs 面积图

| 特性 | 南丁格尔玫瑰图 | 面积图 |
|------|--------|--------|
| 坐标系 | 极坐标 | 笛卡尔坐标 |
| 时间表达 | 周期循环 | 连续趋势 |
| 类别数 | 4-12 | 2-5 |
| 适用 | 周期数据 | 时间序列 |

## 不适用场景

❌ **不推荐使用**
- 精确数值比较 → 用柱状图或条形图
- 层级关系复杂 → 用旭日图或树状图
- 类别过多（>12） → 用柱状图
- 需要精确读数 → 用表格或柱状图
- 负数数据 → 用对比条形图
- 数据点少于4个 → 用柱状图或饼图
- 需要精确的轴标签对齐 → 用柱状图
