# 靶心图 Bullet Chart

图表分类: 比较 Comparison
感知排名: ⭐⭐⭐⭐☆

## 简介

靶心图用深色条表示实际值，用标记线表示目标值，背景范围表示绩效等级（差/中/优）。适合展示单个指标的达成情况，如销售目标达成率、KPI完成度等。

相比柱状图，靶心图更适合展示目标对比；相比仪表盘，靶心图能同时展示多个指标。

## 数据格式

**必需列**

| 列名 | 类型 | 说明 |
|------|------|------|
| label | 文本/类别 | 指标名称 |
| actual | 数值(≥0) | 实际达成值 |
| target | 数值(≥0) | 目标值 |

**可选列**

| 列名 | 类型 | 说明 |
|------|------|------|
| low | 数值(≥0) | 及格线（绩效等级下限） |
| mid | 数值(≥0) | 良好线（绩效等级中限） |
| high | 数值(≥0) | 优秀线（绩效等级上限） |

**示例数据**

| 指标 | 实际 | 目标 | 及格 | 良好 | 优秀 |
|------|------|------|------|------|------|
| 销售额 | 850 | 1000 | 500 | 750 | 1200 |
| 客户满意度 | 8.2 | 8.5 | 7.0 | 8.0 | 9.0 |
| 成本控制 | 45 | 50 | 60 | 55 | 40 |

## 适用场景

✅ **典型场景**
- KPI 达成率：销售额、利润、客户数等
- 绩效评估：员工/部门的多项指标对比
- 目标追踪：项目进度、质量指标、成本控制
- 预算执行：实际支出 vs 预算
- 服务水平：响应时间、完成率等

✅ **使用建议**
1. 指标数控制在 3-20 个为佳
2. 实际值、目标值、范围值应在同一量纲
3. 背景范围应按绩效等级递进（低→中→高）
4. 标签应清晰简洁，避免过长
5. 可按实际值排序便于对比

## 注意事项

- 所有数值必须 ≥ 0（负数应使用对比条形图）
- 背景范围应满足：low ≤ mid ≤ high
- 实际值和目标值应在合理范围内（避免极端值）
- 指标过多（>30）会导致图表过长，建议分页
- 背景范围是可选的，不提供时仅显示实际值和目标值

## 与相似图表的区别

### 靶心图 vs 柱状图

| 特性 | 靶心图 | 柱状图 |
|------|--------|--------|
| 用途 | 目标对比 | 类别对比 |
| 目标线 | 有（标记线） | 无 |
| 背景范围 | 有（绩效等级） | 无 |
| 适用 | KPI/绩效 | 一般比较 |

### 靶心图 vs 仪表盘

| 特性 | 靶心图 | 仪表盘 |
|------|--------|--------|
| 指标数 | 多（3-20） | 单个 |
| 展示方式 | 水平条 | 圆形表盘 |
| 精度 | 高 | 中 |
| 适用 | 多指标对比 | 单指标展示 |

### 靶心图 vs 散点图

| 特性 | 靶心图 | 散点图 |
|------|--------|--------|
| 数据类型 | 单变量 | 双变量 |
| 维度 | 1D | 2D |
| 用途 | 目标达成 | 相关性分析 |
