# 山脊线图 Ridgeline Plot

图表分类: 分布类 DISTRIBUTION
感知排名: ⭐⭐⭐☆☆

## 简介

山脊线图是一系列密度曲线或直方图沿同一水平轴对齐并沿垂直轴略微重叠的图表。通过这种独特的重叠排列，用户可以：

- 观察多个分组的分布形态并进行视觉对比
- 识别分布在不同分组间的变化趋势
- 展示分布的整体模式而不过度强调细节差异

相比小提琴图，山脊线图更适合展示多个分组的分布演变；相比分面直方图，山脊线图通过重叠设计节省空间并增强对比效果。


## 数据格式

需要包含以下字段：

- group（必需）：分类列，表示不同的分组（如行业、时间段、地区）
- value（必需）：数值列，表示要分析的分布变量（如收入、薪资、价格）

或使用**宽格式数据**（自动转换）：

| 行业 | 样本1 | 样本2 | 样本3 | 样本4 | 样本5 |
|------|-------|-------|-------|-------|-------|
| 建筑 | 72.3 | 68.9 | 75.2 | 70.4 | 81 |
| 技术 | 45.2 | 48.6 | 52.1 | 47.5 | 55.3 |
| 零售 | 52.3 | 48.9 | 51.5 | 49.2 | 47.8 |


## 适用场景

山脊线图适用于以下场景：

- 多个分组的分布形态对比（如不同行业的薪资分布）
- 展示分布在分组间的演变趋势（如按时间排序的分布变化）
- 需要在有限空间内展示多个分布的场景
- 强调整体分布模式而非精确数值的演示


## 典型场景

- 不同行业的薪资差距分布对比（如 Guardian 2018 年的性别薪酬差距分析）
- 多个时间段的销售额分布演变
- 不同地区的房价分布变化
- 各产品线的用户评分分布对比


## 使用建议

- 分组数建议 3-15 个，过多会导致图表拥挤
- 按分布特征（如均值、中位数）排序分组，增强视觉效果
- 使用不同颜色区分分组，或用渐变色表示趋势
- 对于分布差异明显的数据效果最佳
- 可配合零线（如性别薪酬差距中的 0% 线）突出对比


## 注意事项

- 宽格式数据会自动转换，第一列作为分组标签，其余列作为数据值
- 分组内数据点建议≥5 个，以获得平滑的密度曲线
- 密度曲线的平滑度受数据量和带宽参数影响
- 当分组数过多时，重叠可能导致可读性下降
- 建议数据不含过多缺失值，否则会影响密度估计


## 与相似图表的区别

### 山脊线图 vs 小提琴图

- 山脊线图：多分组沿垂直轴排列，强调分布演变趋势，节省空间
- 小提琴图：分组并排显示，强调单个分布形态，适合少量分组

### 山脊线图 vs 分面直方图

- 山脊线图：密度曲线平滑、分组重叠、视觉连贯
- 分面直方图：离散柱状、分组独立、便于精确对比

### 山脊线图 vs 面积图

- 山脊线图：展示分布形态，适合非时间序列数据
- 面积图：展示数值累积，适合时间序列或堆叠数据
