# 小提琴图 Violin Chart

图表分类: 分布类 DISTRIBUTION 
感知排名: ⭐⭐⭐⭐☆

## 简介

小提琴图结合了箱线图和密度估计，通过对称的曲线展示数据的完整分布形状。它能够清晰呈现数据的双峰、偏态等特征，使用户可以：

- 观察单个或多个分组的数据分布形态
- 识别分布中的异常值和离群点
- 比较不同分组间的分布差异

相比箱线图，小提琴图更能展示分布的细节；相比直方图，小提琴图更适合多组对比。


## 数据格式

需要包含以下字段：

- y（必需）：数值列，表示要分析的分布变量（如收入、销售额、评分）
- x（可选）：分类列，用于分组对比（如地区、部门、产品线）

## 适用场景

小提琴图适用于以下场景：

- 多组数据分布形态对比（如不同地区的收入分布）
- 识别分布中的双峰或偏态特征
- 展示异常值和离群点的位置
- 需要同时显示分布密度和统计摘要的场景


## 典型场景

- 不同部门员工薪资分布对比
- 多个产品的用户评分分布
- 不同时间段的销售额分布变化
- 各地区房价分布的形态分析


## 使用建议

- 每组数据建议≥10个数据点，以获得平滑的密度曲线
- 当分组数>5时，考虑改用分面（facet）或改为箱线图以避免拥挤
- 配合箱线（inner='box'）显示四分位数，增强可读性
- 对于小样本数据，可改用箱线图+散点图


## 注意事项

- 宽格式数据会自动转换，第一列作为分组标签，其余列作为数据值
- 离群点默认显示为单个点，可通过 `points` 参数调整
- 当分组内数据点过少（<5）时，密度曲线可能不稳定
- 建议数据不含过多缺失值，否则会影响密度估计


## 与相似图表的区别

### 小提琴图 vs 箱线图

- 小提琴图：展示完整分布形态，适合识别双峰、偏态等特征
- 箱线图：简洁展示四分位数和离群点，适合快速对比

### 小提琴图 vs 直方图

- 小提琴图：多组对比、分布密度平滑、适合分组数据
- 直方图：单组分布、离散柱状、适合展示频数分布
