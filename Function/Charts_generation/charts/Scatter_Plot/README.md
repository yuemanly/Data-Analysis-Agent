# 散点图 Scatter Plot

图表分类: 关系 Relationship  
感知排名: ⭐⭐⭐⭐⭐

## 简介

散点图是展示两个数值变量之间关系的最常见可视化方式。一个变量绘制在水平轴上，另一个变量绘制在垂直轴上。每个观测值在创建的空间中绘制为一个点。

与柱状图不同，散点图的轴不必从零开始，特别是当零不是数据的可能值时。散点图可以帮助读者发现两个变量是否相关：
- 正相关：两个变量同时增大或减小（向右上方移动）
- 负相关：两个变量反向变化（向右下方移动）
- 无相关：没有明显的关系

## 数据格式

需要包含以下字段：

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| x | 数值 | ✅ | X轴数据，通常为自变量 |
| y | 数值 | ✅ | Y轴数据，通常为因变量 |
| size | 数值 | ❌ | 气泡大小列，表示第三个维度（可选） |
| color | 文本/数值 | ❌ | 颜色分组列，用于区分不同类别（可选） |

### 典型数据结构

| 面积 | 价格 | 房间数 | 区域 |
|------|------|--------|------|
| 50   | 300  | 2      | 北区 |
| 75   | 450  | 3      | 南区 |
| 60   | 380  | 2      | 北区 |
| 90   | 520  | 4      | 南区 |
| 55   | 320  | 2      | 中区 |

**字段映射示例：**
- x: 面积
- y: 价格
- size: 房间数（可选）
- color: 区域（可选）

## 适用场景

- 两变量相关关系：身高vs体重、广告投入vs销售额、GDP vs碳排放、生育率vs寿命
- 分组聚类分析：不同地区的房价分布、不同产品的销售表现、不同国家的发展指标
- 异常值检测：发现离群点和异常数据
- 第三维度展示：用气泡大小表示人口、用颜色表示地区

## 典型场景

- 房价分析：展示房屋面积与价格的关系，用房间数表示气泡大小，用区域区分颜色
- 销售分析：展示广告投入与销售额的关系，用不同产品线区分颜色
- 环境分析：展示GDP与碳排放的关系，用国家人口表示气泡大小
- 国际发展：展示生育率与寿命的关系（Gapminder风格），用国家人口表示气泡大小，用地区区分颜色

## 使用建议

- 数据点建议 ≥ 5 个，以展示明显的关系
- x和y列必须为连续数值类型
- 透明度处理：当数据点重叠时，使用透明度（opacity参数）使重叠点可见
  - 默认透明度0.7，可根据数据密度调整
  - 大量重叠点（>1000）考虑使用hexbin热力图替代
- 趋势线：启用trendline选项显示线性回归线
  - 帮助读者更清晰地看到相关性方向和强度
  - 显示R²值表示拟合度
- 分组颜色：使用color列区分不同类别
  - 自动分配不同颜色
  - 支持最多10个分组
- 气泡大小：使用size列表示第三维度
  - 自动线性映射到5-30像素范围
  - 如不提供，所有点大小统一

## 注意事项

- 缺失值会被自动删除
- x和y列必须可转换为数值类型
- 少于2个有效点会导致生成失败
- 大量重叠点会影响视觉效果，可考虑添加透明度或使用热力图
- 轴不必从零开始，特别是当零不是数据的可能值时

## 与相似图表的区别

### 散点图 vs 折线图

- 散点图：展示两个变量的关系，强调点的分布和相关性
- 折线图：展示时间序列数据，强调趋势变化

### 散点图 vs 气泡图

- 散点图：用size表示第三维度，用color表示分组，支持趋势线
- 气泡图：专门用气泡大小表示第三维度，通常不用color分组

### 散点图 vs 热力图

- 散点图：适合点数较少（<1000）的数据，每个点清晰可见
- 热力图（hexbin）：适合点数众多（>1000）的数据，避免重叠
