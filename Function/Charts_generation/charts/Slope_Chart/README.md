# 斜率图

图表分类: 变化 Change  
感知排名: ⭐⭐⭐☆☆

## 简介

斜率图通过连接两个时间点的数据，用线条的斜率直观展示实体在时间维度上的变化幅度和方向。它特别适合对比两个时间点间的排名或数值变化：

- 快速识别增长与下降的实体
- 清晰展示变化幅度的差异
- 便于排名变化的对比分析

相比折线图，斜率图更聚焦于两个关键时间点的对比；相比柱状图，斜率图能更直观地表现变化趋势。

## 数据格式

需要包含以下字段：

- group：实体名称（国家、公司、产品等），文本类型
- start：起始时间点的数值，数值类型
- end：终止时间点的数值，数值类型

典型数据结构：

| 国家 | 2010年GDP | 2020年GDP |
|------|----------|----------|
| 中国 | 5.9 | 14.7 |
| 美国 | 10.3 | 20.9 |
| 日本 | 5.5 | 5.1 |
| 德国 | 3.4 | 3.8 |

## 适用场景

斜率图适用于以下场景：

- 两个时间点间的排名变化对比
- 实体增长或下降幅度的对比分析
- 市场份额、销售额、排名等指标的变化展示
- 强调变化方向和幅度差异的数据呈现

## 典型场景

- 国家GDP在十年间的增长对比
- 企业市场份额在两个季度间的变化
- 产品销售排名在年初和年末的变化
- 学生成绩在两次考试间的进退

## 使用建议

- 实体数量建议不超过 30 个，过多会导致图表拥挤
- 变化幅度用颜色编码：绿色表示增长，红色表示下降
- 数据会按变化幅度自动排序，便于快速识别最大变化

## 注意事项

- 仅支持两个时间点的对比，多于两个时间点应使用折线图
- 如果实体过多（>30），建议筛选或分组展示
- 确保 start 和 end 列都是数值类型，避免文本混入
- 缺失数据会导致该实体无法绘制

## 与相似图表的区别

### 斜率图 vs 折线图

- 斜率图：聚焦两个时间点，强调变化幅度和方向
- 折线图：展示多个时间点的连续变化趋势

### 斜率图 vs 柱状图

- 斜率图：用线条斜率编码变化，便于对比多个实体的变化方向
- 柱状图：用柱子高度编码单个时间点的数值，适合单时间点对比

### 斜率图 vs 凹凸图

- 斜率图：强调数值变化的幅度
- 凹凸图：强调排名变化的顺序
