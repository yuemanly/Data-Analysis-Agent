# 柱状图 Bar Chart

图表分类: 对比 COMPARING
感知排名: ⭐⭐⭐⭐⭐

## 简介

柱状图通过矩形的高度（或宽度）编码数值大小，是最常用的比较图表。适合展示不同类别的数值对比，如各国GDP、各产品销量等。

相比折线图，柱状图更适合离散类别的比较；相比饼图，柱状图能准确表达数值差异。

## 数据格式

**必需列**

| 列名 | 类型 | 说明 |
|------|------|------|
| x | 文本/类别 | 类别标签（横轴） |
| y | 数值(≥0) | 数值大小（纵轴） |

**可选列**

| 列名 | 类型 | 说明 |
|------|------|------|
| series | 文本/类别 | 分组标签（用于分组柱状图） |
| color | 文本 | 自定义颜色 |

**示例数据**

| 国家 | GDP | 年份 |
|------|-----|------|
| 中国 | 17.9 | 2023 |
| 美国 | 27.4 | 2023 |
| 日本 | 4.2 | 2023 |
| 德国 | 4.3 | 2023 |

## 适用场景

✅ **典型场景**
- 跨类别数值比较：各国GDP、各产品销量、各部门收入
- 两时期截面对比：去年 vs 今年销售额
- 排名展示：销售额排名前10的产品
- 成分对比：各类别的占比（配合堆叠柱状图）
- 目标达成率：实际 vs 目标（配合靶心图）

✅ **使用建议**
1. 类别数控制在 3-15 个为佳（过多难以阅读）
2. 数值必须非负，y 轴从零开始
3. 按数值大小排序便于对比
4. 颜色应保持一致或按分组区分
5. 标签应清晰简洁，避免重叠

## 注意事项

- 数值必须 ≥ 0（负数应使用对比条形图）
- y 轴必须从零开始，禁止截断（避免误导）
- 类别过多（>30）会导致标签重叠，建议分页或过滤
- 柱子宽度应适中，避免过宽或过窄
- 同一图表中的柱子应保持相同宽度

## 与相似图表的区别

### 柱状图 vs 折线图

| 特性 | 柱状图 | 折线图 |
|------|--------|--------|
| 数据类型 | 离散类别 | 连续时间序列 |
| 用途 | 类别间比较 | 趋势展示 |
| 精度 | 高（精确值） | 中（趋势为主） |
| 适用 | 截面数据 | 时间序列 |

### 柱状图 vs 饼图

| 特性 | 柱状图 | 饼图 |
|------|--------|------|
| 数据类型 | 绝对值 | 相对比例 |
| 精度 | 高 | 低（难以比较） |
| 类别数 | 多（3-30） | 少（3-5） |
| 用途 | 数值对比 | 占比展示 |

### 柱状图 vs 散点图

| 特性 | 柱状图 | 散点图 |
|------|--------|--------|
| 数据类型 | 单变量 | 双变量 |
| 维度 | 1D（高度） | 2D（x,y） |
| 用途 | 单一指标对比 | 相关性分析 |

### 柱状图 vs 热力图

| 特性 | 柱状图 | 热力图 |
|------|--------|--------|
| 数据类型 | 1D 类别 | 2D 矩阵 |
| 维度 | 1D | 2D |
| 用途 | 单一指标 | 多维数据 |
