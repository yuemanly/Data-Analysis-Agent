# 桑基图 Sankey Diagram

图表分类: 流向 Flow
感知排名: ⭐⭐⭐⭐

## 简介

桑基图用带宽度的箭头表示从起点到终点的流量、转移或流动。箭头的宽度与流量大小成正比，适合展示能源流动、资金转移、用户流向等多对多的流向关系。

## 数据格式

**必需列**

| 列名 | 类型 | 说明 |
|------|------|------|
| source | 文本/类别 | 流量起点（节点名称） |
| target | 文本/类别 | 流量终点（节点名称） |
| value | 数值(≥0) | 流量大小（箭头宽度） |

**示例数据**

| source | target | value |
|--------|--------|-------|
| 电厂A | 城市1 | 100 |
| 电厂A | 城市2 | 80 |
| 电厂B | 城市2 | 120 |
| 电厂B | 城市3 | 60 |

## 适用场景

✅ **典型场景**
- 能源流向：电厂→城市、油田→炼厂→加油站
- 资金流向：部门→项目、收入→支出
- 人员流动：地区→地区、学校→工作
- 物流网络：仓库→配送中心→门店
- 用户转化：渠道→注册→付费→留存

✅ **使用建议**
1. 节点数控制在 5-20 个为佳（过多会混乱）
2. 流量值应为正数，0 值会被忽略
3. 自动检测 source/target/value 列，也可手动指定
4. 支持自环（同一节点的内部流动）

## 注意事项

- 重复边会自动合并（同 source→target 的多行会求和）
- 孤立节点（无入度或出度）不会显示
- 流量值为 0 或 NaN 的行会被过滤
- 节点颜色自动分配，可通过 options 自定义

## 与相似图表的区别

### 桑基图 vs 弦图

| 特性 | 桑基图 | 弦图 |
|------|--------|------|
| 流向 | 单向（A→B） | 双向（A↔B） |
| 用途 | 流量/转移 | 关系强度 |
| 节点排列 | 左右两列 | 圆形 |
| 适用 | 线性流程 | 循环关系 |

### 桑基图 vs 网络图

| 特性 | 桑基图 | 网络图 |
|------|--------|-------|
| 边权重 | 必需（流量） | 可选 |
| 布局 | 固定（层级） | 自由（力导向） |
| 用途 | 流向展示 | 关系探索 |
| 复杂度 | 简洁 | 灵活 |
