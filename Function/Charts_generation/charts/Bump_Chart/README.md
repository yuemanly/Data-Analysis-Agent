# 凹凸图

图表分类: 排名
感知排名: ⭐⭐⭐⭐☆

## 简介

凹凸图用于展示多个实体的排名随时间的变化，通过连接线展示排名变化趋势。它通过相对排名而非绝对值来展示数据，使用户可以直观观察：

- 各实体排名的上升或下降趋势
- 实体间排名的相对位置变化
- 黑马（快速上升）和掉队者（快速下降）的识别

相比折线图，凹凸图更适合展示相对排名而非绝对值；相比排序条形图，凹凸图更适合展示排名随时间的动态变化。


## 数据格式

需要包含以下字段：

- x 列（时间）：时间序列数据，如年份、月份、周次等
- y 列（排名/分数）：排名数值或评分，数值越小排名越靠前
- group 列（实体名称）：实体标识，如国家、品牌、球队等

典型数据结构：

| 年份 | 国家 | 排名 |
|------|------|------|
| 2010 | 美国 | 1 |
| 2010 | 德国 | 3 |
| 2010 | 法国 | 5 |
| 2015 | 美国 | 2 |
| 2015 | 德国 | 1 |
| 2015 | 法国 | 4 |


## 适用场景

凹凸图适用于以下场景：

- 医疗支出排名变化：展示各国医疗支出占 GDP 比例的排名变化
- 销售排名变化：展示各产品或地区销售排名的变化
- 搜索热度排名：展示各关键词搜索指数排名的变化
- 体育赛事排名：展示各队伍在赛季中的排名变化


## 典型场景

- 全球 GDP 排名变化：展示各国 GDP 排名在 10 年间的变化
- 品牌市场份额排名：展示各品牌市场份额排名的季度变化
- 应用下载排名：展示各应用在应用商店排名的周度变化
- 政治民调排名：展示各候选人民调支持率排名的变化


## 使用建议

- 实体数建议 ≤ 15 个，避免线条交叉过多导致难以阅读
- 使用排名（rank）而非绝对值，以突出相对位置变化
- 可通过 `highlight` 参数高亮感兴趣的实体，使其更突出
- 时间点建议 ≥ 3 个，以展示明显的排名变化趋势


## 注意事项

- 缺失值会被自动删除，可能导致某些实体的线条不连续
- y 列必须可转换为数值类型，否则会被跳过
- 如果 y 列名称包含"rank"或"排名"，y 轴会自动反转（排名越小越靠前）
- 实体数过多时会产生警告，但仍会生成图表


## 与相似图表的区别

### 凹凸图 vs 折线图

- 凹凸图：展示相对排名，线条不相交，强调排名变化而非绝对值差异
- 折线图：展示绝对值，线条可相交，强调数值大小和变化幅度

### 凹凸图 vs 排序条形图

- 凹凸图：展示排名随时间的动态变化，适合多个时间点
- 排序条形图：展示单个时间点的排名，适合静态排序展示

### 凹凸图 vs 流图

- 凹凸图：线条宽度固定，强调排名变化
- 流图：线条宽度代表数值大小，同时展示排名和数值变化
