# 箱线图 Box-and-Whisker Plot

图表分类: 分布类 DISTRIBUTION  
感知排名: ⭐⭐⭐⭐☆

## 简介

箱线图通过中位数、四分位数和异常值的位置展示数据分布特征。它用矩形（箱）表示中间50%的数据，用线条（须）表示数据范围，用点表示异常值。相比直方图，箱线图更紧凑；相比小提琴图，箱线图更简洁。

它通过以下方式帮助用户观察数据：

- 快速比较多个组别的分布形态（中位数、离散度）
- 识别异常值和极端值
- 判断数据的对称性和偏态


## 数据格式

需要包含以下字段：

- y（必需）：数值列，要展示分布的变量（如价格、销售额、评分）
- x（可选）：分类列，用于分组对比（如产品类别、地区、时间段）

## 适用场景

箱线图适用于以下场景：

- 多个组别的分布对比（如不同产品的价格分布）
- 发现异常值和极端数据点
- 评估数据的离散程度和对称性
- 时间序列中的分布变化（如按月份分组）


## 典型场景

- 不同城市房价分布对比
- 多个供应商交货时间的稳定性评估
- 各部门员工薪资分布
- 产品质量指标的波动范围


## 使用建议

- 每组数据建议≥5个数据点，以获得有意义的四分位数
- 分组数建议≤10个，过多会导致图表拥挤
- 异常值由IQR×1.5法则自动识别（超出此范围的点单独显示）
- 可配合 `strip=True` 参数显示原始数据点，增强透明度


## 注意事项

- 宽格式数据会自动转换，第一列作为分组标签，其余列作为数值
- 空值会被自动忽略
- 异常值判定标准：Q1 - 1.5×IQR 或 Q3 + 1.5×IQR
- 数据量过少（<5/组）时建议改用数据摘要表格


## 与相似图表的区别

### 箱线图 vs 小提琴图

- 箱线图：展示四分位数、中位数、异常值，结构清晰，适合快速对比
- 小提琴图：展示完整分布密度，细节丰富，适合深入分析

### 箱线图 vs 直方图

- 箱线图：多组对比首选，紧凑高效
- 直方图：单组分布细节展示，适合观察分布形状
