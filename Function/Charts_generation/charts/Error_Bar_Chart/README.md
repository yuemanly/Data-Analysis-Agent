# 误差条形图

图表分类: 分布 Distribution  
感知排名: ⭐⭐⭐⭐☆

## 简介

误差条形图通过在柱状图上添加误差条，展示分组数据的中位数及其四分位数范围，使用户可以：

- 观察每组数据的中位数
- 了解数据的分布范围（Q25-Q75）
- 比较不同组的数据分布特征
- 快速识别异常组或离群值

系统会自动计算四分位数，无需用户手动输入。


## 数据格式

需要包含以下字段：

- Label（必需）：分组类别（如行业、地区、产品等）
- x（必需）：原始数值（系统自动计算 Q25、Q50、Q75）

示例：

| 行业 | 收入 |
|------|------|
| 金融 | 85000 |
| 金融 | 92000 |
| 金融 | 78000 |
| 科技 | 75000 |
| 科技 | 82000 |
| 科技 | 68000 |
| 制造 | 65000 |
| 制造 | 72000 |
| 制造 | 58000 |

系统会自动按行业分组，计算每组的四分位数。


## 适用场景

- 展示分组数据的分布特征
- 比较不同组的中位数和离散程度
- 识别数据的异常值和离群点
- 评估不同类别的数据变异性
- 展示原始数据的统计特征


## 典型场景

- 各行业员工收入分布
- 不同地区的房价分布
- 多个产品的质量指标分布
- 不同时间段的销售额分布
- 各部门的绩效评分分布


## 使用建议

- 数据应包含足够的样本量（每组建议 ≥ 10 个数据点）
- 系统自动计算 Q25（下界）、Q50（中位数）、Q75（上界）
- 误差条表示四分位数范围，不是标准差或置信区间
- 悬停提示会显示中位数、Q25、Q75 和样本数
- 当样本量不足时，四分位数可能不稳定


## 注意事项

- 数据必须包含 x（分组）和 y（数值）两列
- 系统会自动按 x 分组，计算每组的四分位数
- 空值和非数值会被自动过滤
- 每组至少需要 1 个数据点（建议 ≥ 10 个）
- 四分位数计算使用 numpy 的默认方法（线性插值）


## 与相似图表的区别

### 误差条形图 vs 箱线图

- 误差条形图：展示中位数和四分位数范围，简洁直观
- 箱线图：展示四分位数、中位数、异常值等更多细节

### 误差条形图 vs 小提琴图

- 误差条形图：简洁，易于比较多个类别
- 小提琴图：展示完整的分布形状，更详细但更复杂

### 误差条形图 vs 条纹图

- 误差条形图：展示汇总统计（中位数和四分位数）
- 条纹图：展示原始数据点，保留所有细节

### 误差条形图 vs 柱状图

- 误差条形图：展示分布范围，适合展示变异性
- 柱状图：仅展示单一数值，适合展示绝对值
